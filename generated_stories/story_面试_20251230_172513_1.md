# 剧本生成：面试篇

这里是为您定制的 3 个【面试】主题爆款反转短剧剧本。

---

### 剧本标题：[面试] - [反购公司]
**角色**: 
* **角色A (HR主管)**: 戴金丝眼镜，眼神轻蔑，手里拿着一杯廉价咖啡。
* **角色B (求职者)**: 穿着人字拖、大裤衩，一脸没睡醒的样子。

| 时间 | 画面/动作/特效 | 台词 (对白) | BGM/音效 |
| :--- | :--- | :--- | :--- |
| **0-5s** | [特写] A把简历撕成两半，扔在B脸上。<br>[动作] A轻蔑地吹了吹手指。 | **A**: 月薪三千你还嫌少？也不撒泡尿照照镜子，门在那边，滚！ | 嘈杂办公室背景音<br>音效：嘶啦 (撕纸声) |
| **5-10s** | [中景] B打了个哈欠，慢悠悠从裤裆里掏出一份加盖钢印的红头文件，拍在桌上。<br>[特效] 文件落地处炸出“收购合同”四个金色大字。 | **B**: 三千是不多，但这公司五千万卖不卖？我买了。 | 音效：Duang!!<br>音效：支付宝到账五千万元 |
| **10-15s** | [特写] A眼镜滑落，瞳孔地震。<br>[动作] A瞬间滑跪到B面前，用撕碎的简历给自己擦鞋，脸上堆满油腻的笑。 | **A**: 哎哟我的董事长爸爸！刚才那是为了考验您的抗压能力！我是您失散多年的看门狗啊！ | BGM骤停 -> <br>土味DJ《好运来》加速版 |
| **结尾** | [定格] 画面定格在A抱着B大腿，脸贴在人字拖上的瞬间。 | **屏幕花字**: 恭迎董事长 | 全曲高潮段落 |

### 剧本标题：[面试] - [房东驾到]
**角色**: 
* **角色A (势利眼面试官)**: 妆容精致，鼻孔朝天，拿着激光笔指指点点。
* **角色B (朴素大叔)**: 穿着背心，腰间挂着一大串钥匙，手里提着蛇皮袋。

| 时间 | 画面/动作/特效 | 台词 (对白) | BGM/音效 |
| :--- | :--- | :--- | :--- |
| **0-5s** | [近景] A用笔敲击桌面，一脸嫌弃地看着B。<br>[动作] A捏着鼻子仿佛闻到了异味。 | **A**: 大爷，我们要的是精英，不是收破烂的！您走错地儿了吧？我们要下班了！ | 紧张/压迫感音效 |
| **5-10s** | [特写] B憨厚一笑，解下腰间巨大的一串钥匙，“哗啦”一声砸在桌上。<br>[特效] 钥匙发出耀眼金光，整个屏幕震动。 | **B**: 没走错，我是来这栋楼收租的。顺便问问，你们公司欠半年的房租啥时候交？ | 音效：重金属撞击声<br>音效：哇哦~ (惊叹声) |
| **10-15s** | [中景] A手中的笔吓掉了，表情从傲慢瞬间变成极度惊恐。<br>[动作] A立刻端起桌上的茶水，双手奉上，卑微鞠躬90度。 | **A**: 房东爷爷！您看您，来就来嘛还带什么钥匙！这茶刚泡好，您润润喉，房租我这就去卖肾给您凑！ | BGM骤停 -> <br>欢快唢呐《喜庆日子》 |
| **结尾** | [定格] 画面定格在A卑微敬茶，B淡定剔牙。 | **屏幕花字**: 楼主好人 | 唢呐高音 |

### 剧本标题：[面试] - [顶级猎物]
**角色**: 
* **角色A (傲慢老板)**: 梳着油头，抽着雪茄，把脚翘在桌子上。
* **角色B (天才程序员)**: 戴厚底眼镜，格子衫，抱着破旧笔记本电脑，看起来木讷。

| 时间 | 画面/动作/特效 | 台词 (对白) | BGM/音效 |
| :--- | :--- | :--- | :--- |
| **0-5s** | [特写] A向B吐了一口烟圈。<br>[动作] A挥挥手像赶苍蝇一样。 | **A**: 只要985硕博，你个本科生凑什么热闹？这行代码你看得懂吗？回去养猪吧！ | 嘲讽/嘘声 |
| **5-10s** | [特写] B的手机突然响起，按下免提。<br>[特效] 手机屏幕投射出“马斯克” (Musk) 的头像和通话界面。 | **电话音(英文+中字)**: Hey bro! 年薪一亿美金来我这，私人飞机已经在楼顶等你接驾了！ | 音效：IPhone经典铃声<br>音效：飞机起飞轰鸣声 |
| **10-15s** | [近景] A嘴里的雪茄掉在裤裆上，烫得一激灵。<br>[动作] A直接扑过去抱住B要离开的腿，死命往回拖，哭得一把鼻涕一把泪。 | **A**: 义父！义父别走！刚才是我有眼无珠！我把公司过户给您！求您带我飞啊义父！ | BGM骤停 -> <br>悲情/激昂神曲《雪花飘飘》 |
| **结尾** | [定格] 画面定格在A抱腿痛哭，B一脸冷漠看向远方。 | **屏幕花字**: 义父受孩儿一拜 | BGM高潮 |
